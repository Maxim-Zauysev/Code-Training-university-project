# Code Training README

Code Training - это приложение для тренировки навыков быстрого набора кода. Оно предназначено для программистов всех уровней, желающих улучшить свою скорость и точность при написании кода.

## Особенности

- **Регистрация и вход:** Пользователи могут регистрироваться и входить в свои аккаунты для отслеживания прогресса.
- **Выбор языка программирования:** Поддержка различных языков программирования с возможностью выбора пользователем.
- **Выбор уровня сложности:** Пользователи могут выбирать уровень сложности генерируемого кода.
- **Подсветка синтаксиса:** Подсветка синтаксиса для улучшения читаемости и удобства написания кода.
- **Проверка на ошибки:** Система подсветки ошибок при сравнении написанного кода с исходным.
- **Статистика и результаты:** Отображение статистики по написанному коду, включая время выполнения и процент совпадения с исходным кодом.
- **Просмотр статистики:** Возможность просмотра как собственной статистики, так и статистики других пользователей по конкретному коду.
- **Добавление новых языков и кода:** Возможность расширения базы языков программирования и примеров кода.

## Технологии

- **SceneBuilder:** Инструмент для проектирования пользовательского интерфейса.
- **JavaFX:** Библиотека для создания настольных приложений с графическим интерфейсом.
- **BCrypt:** Библиотека для хеширования паролей и их безопасного хранения в базе данных.

## Как начать использовать

Для начала работы с приложением:

1. Установите Java и JavaFX на ваш компьютер.
2. Клонируйте репозиторий на свой локальный компьютер.
3. Откройте проект в вашей среде разработки и настройте подключение к базе данных в классе DatabaseManager. 
4. Запустите приложение.
